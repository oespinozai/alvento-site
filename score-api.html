What You Get

One POST request returns five scored areas with specific findings and actionable fixes:

  • Google Business Profile — photos, hours, phone, categories, completeness
  • SEO — mobile speed, title tags, schema markup, meta descriptions
  • Reviews — count, rating, recency, response rate
  • Citations — directory presence, NAP consistency
  • Competitors — relative position and gap analysis

Plus a prioritised list of the top 5 fixes, ordered by impact.
